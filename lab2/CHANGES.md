# Изменения в Лабораторной работе №2 (по сравнению с ЛР №1)

1. **Обновлен Dockerfile**:
   - Добавлена установка пакетов ОС `procps` и `default-jre` (требуется для работы Spark) от имени пользователя `root`.
   - Добавлена установка Python-библиотек `apache-airflow-providers-apache-spark==4.1.1` и `pyspark==3.5.0` от имени пользователя `airflow`.
   - Добавлено копирование новой директории `spark` внутрь образа Airflow.

2. **Обновлен docker-compose.yml**:
   - Добавлены новые сервисы для Apache Spark: `spark-master` и `spark-worker`.
   - Проброшены порты для доступа к веб-интерфейсу Spark (4040 для мастера) и порту подключения (7077).
   - В общих настройках volumes для `airflow-*` добавлена директория `./spark`.

3. **Добавлены скрипты Spark**:
   - Создана директория `spark` и добавлен скрипт `my_script.py`. В скрипте инициализируется `SparkSession`, подключающийся к `spark-master:7077`, и выполняется простая Spark-задача (создание и вывод DataFrame).

4. **Добавлен новый DAG**:
   - Создан файл `dags/spark_dag.py`, в котором используется `SparkSubmitOperator` для запуска скрипта `my_script.py` в кластере Spark через Airflow.
